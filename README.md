
# Spotify Clone - Frontend

## Demo

https://spotify-clone-rouge-eight.vercel.app/

## Technologies Used

- React.js
- Tailwind
- Redux


## Features

- Home,Search,Collection,Profile pages
- Volume on/off
- Songs audio listening 
- Song pause-advance
- Responsive Design
